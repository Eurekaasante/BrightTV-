📺 BrightTV Customer Value Management (CVM) Insights Project
Purpose
Growth is never accidental. It is engineered through clarity of intent, disciplined execution, and deep respect for the customer.
This project represents a deliberate, data‑driven effort to support BrightTV’s strategic ambition to grow its subscription base during the current financial year. It was undertaken to empower the Customer Value Management (CVM) team with insights that do not merely describe what has happened, but inform what must happen next.
At its core, this work seeks to answer a simple but profound question:
How do we create enduring value for our customers while building a sustainable, growing platform?

Strategic Objective
The primary objective of this initiative is to enable BrightTV to:

Deepen customer engagement through increased content consumption
Build consistent viewing habits, particularly during low‑engagement periods
Understand the true drivers of customer choice and behaviour
Grow the user base in a manner that is sustainable, purposeful, and customer‑centric


Data Foundation
Data Source and Structure
The analysis is grounded in data provided in CSV format, originally organised into two distinct worksheets, each representing a critical dimension of the BrightTV ecosystem.
These worksheets were separated and uploaded into the Databricks platform as two structured analytical tables:


User Profile Table

Subscriber and account‑level attributes
Subscription characteristics and metadata



Viewer Transactions Table

Session‑level viewing data
One record per consumption session, enabling precise behavioural analysis



This deliberate structuring ensures analytical integrity, scalability, and clarity.

Time and Consumption Integrity

All timestamps were originally supplied in UTC
For contextual accuracy and local relevance, time variables were converted to
South African Standard Time (SAST, UTC +2)
Consumption is assessed per session, recognising that each viewing moment represents a conscious customer choice

Methodology
This initiative followed a disciplined and transparent methodology:
1. Data Preparation

Ingestion of CSV files into Databricks
Separation of source worksheets into analytical tables
Validation of structure, quality, and consistency
Timezone conversion to ensure relevance and accuracy

2. Exploratory Data Analysis

SQL‑based analysis in Databricks
Focus areas included:

Time‑of‑day and day‑of‑week usage trends
Session frequency and duration patterns
Consumption distribution across the user base
Identification of peak and under‑utilised periods



3. Insight Generation

Aggregated datasets exported into Excel
Visual exploration to surface trends, disparities, and opportunities

4. Business Interpretation

Translation of data signals into meaningful business narratives
Alignment of findings to Customer Value Management objectives

5. Recommendation Formulation

Content strategies designed for low‑consumption periods
Engagement and personalisation initiatives rooted in customer behaviour
Practical approaches to acquisition and retention

6. Storytelling and Delivery

Insight narrative shaped through Miro and Canva
Final presentation delivered in a leadership‑ready PowerPoint format ha focused as enlisted below:-

Analytical Focus Areas

User and usage behaviour trends
Factors influencing consumption decisions
Optimisation of low‑engagement days
Content performance patterns
Sustainable user base growth opportunities
